# tp-vuejs

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
## Docker
### Build
```sh
docker-compose up --build -d
```
### Stop
```sh
docker-compose down
```
